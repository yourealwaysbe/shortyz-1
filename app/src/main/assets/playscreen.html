Play Board Help

On this screen you solve your puzzle!

  • Keyboard behaviour can be configured in the settings.
  • Tap clue line at the top to show/hide clues panel.
  • Long-press clue line to open clues page.
  • Long-press clue on board to open notes on clue.

Pressing the Menu button provides the following options:

  • Clues
    Takes you to a screen where you can browse the across and down clues for the whole puzzle.
  • Notes
    Takes you to a screen where you can make notes on the clue and solve anagrams.
  • Zoom
    Control the zoom of the play board.
  • Clue Text Size
    Alter the size of the clues.
  • Show Errors
    Highlights unsolved or incorrect boxes in red. (May be disabled if no answer key is available)
  • Scratch Mode
    Edit clue scratch (see notes) directly. Configure show scratch on board in settings to use this best.
  • Reveal
    Presents the option to reveal the correct solution to a box, word or the whole puzzle. Once revealed "cheated" boxes have a pink background. (May be disabled if no answer key is available)
  • Info
    Shows information about the puzzle you are playing.
  • Support Puzzle Source
    Available if known. Opens the webpage of the source so you can offer your support (via e.g. newspaper subscriptions).
  • Help
    Displays this screen
  • Settings
    Takes you to the application settings screen.
